1. Clone bootstrap
2. Go to branch v4-dev
3. git flow init
4. Download node
https://nodejs.org/en/
5. Install grunt
sudo npm install -g grunt-cli
6. Go to /bootstrap and do 
sudo npm install
7. Install Ruby
brew install ruby
8. Install bundle
gem install bundler
9. bundle install
10. bundle exec jekyll serve
11. http://localhost:9001
